#  Projeto Login 🔐

Uma interface de **tela de login responsiva** desenvolvida utilizando **HTML e CSS**, com foco em layout moderno e adaptação para diferentes tamanhos de tela.

Este projeto foi criado com o objetivo de praticar **desenvolvimento front-end**, principalmente estruturação de páginas, estilização com CSS e criação de formulários.

---

##  Tecnologias Utilizadas 🚀

- HTML5
- CSS3
- Google Fonts
- Material Symbols

---

## 📂 Estrutura do Projeto

```
projeto-login
│
├── index.html
├── estilos
│   └── style.css
└── imagens
```

---

## 💻 Como executar o projeto

1. Clone o repositório

```bash
git clone https://github.com/rafaellimatech/projeto-login.git
```

2. Entre na pasta do projeto

```bash
cd projeto-login
```

3. Abra o arquivo **index.html** no navegador.
