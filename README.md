# Projeto Login

Uma interface de tela de login responsiva desenvolvida utilizando HTML e CSS, com foco em layout moderno, organização de formulário e adaptação para diferentes tamanhos de tela.

Este projeto foi criado com o objetivo de praticar desenvolvimento front-end, principalmente estruturação de páginas, estilização com CSS e criação de formulários de autenticação.

🚀 # Tecnologias Utilizadas

HTML5

CSS3

Google Fonts

Material Symbols

💻 Como executar o projeto

Clone o repositório

git clone https://github.com/rafaellimatech/projeto-login.git

Entre na pasta do projeto

cd projeto-login

Abra o arquivo index.html no navegador.
